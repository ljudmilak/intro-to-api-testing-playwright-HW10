## PUT order
| # | Сценарий | Тест данные | Код |
| --- | --- | --- | --- | 
| 1 | Заказ успешно обновлен | orderId = { 1 .. 10 }, валидный 16-цифр API key, body не пустой | 200 |
| 2 | Неверный API key | API key отсутствует, невалидный или не 16 цифр | 401 |
| 3 | body пустой | пустой body запроса | 404 (реальный сервер возвращает код 200) |

## DELETE order
| # | Сценарий | Тест данные | Код |
| --- | --- | --- | --- | 
| 1 | Заказ успешно удален | orderId = { 1 .. 10 }, валидный 16-цифр API key | 204 |
| 2 | Неверный API key | API key отсутствует, невалидный или не 16 цифр | 401 |

## GET order

| --- | --- | --- | --- |

| 2 | username или password отсутсвует | пустой username или password  | 500 |
